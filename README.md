## Como executar o projeto
<!-- criar banco de dados -->
npm install

<!-- criar banco de dados -->
npm run start

<!-- criar banco de dados -->
npx sequelize-cli db:create

<!-- migrate no banco -->
npx sequelize-cli db:migrate
